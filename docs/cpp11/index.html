C++11 新特性概览

C++11 是现代 C++ 的里程碑，引入了移动语义、Lambda、智能指针、并发支持等重大特性，彻底改变了 C++ 编程方式。

C++11 的重要性

C++11 是现代 C++ 的起点。掌握 C++11 是学习后续所有 C++ 标准版本的基础。本章包含 33 个独立可编译示例，覆盖语言核心、模板、标准库、并发等所有方面。
